# Sales Overview Dashboard (2021–2023)

## Project Overview
The **Bev and Co Sales Overview Dashboard** is an interactive Business Intelligence project designed to analyze sales performance across multiple dimensions including products, categories, customers, and time periods.

The dashboard provides a comprehensive view of key metrics such as total revenue, units sold, customer segmentation, pricing comparisons, and discount distributions.

This project demonstrates how data visualization tools can transform raw sales data into meaningful business insights.

---


## Key Features

### KPI Metrics
- **Total Sales ($)** – Overall revenue generated
- **Total Units Sold** – Total quantity of products sold

### Visual Analytics

The dashboard includes the following visualizations:

1. **Total Category Sales**
   - Displays revenue distribution across product categories.

2. **Customer Sales Distribution**
   - Shows sales split between **B2B** and **B2C** customers.

3. **Total Product Sales**
   - Compares sales performance across individual products.

4. **Discount (%) by Category**
   - Displays discount distribution among categories.

5. **Total Sales per Quarter**
   - Highlights quarterly sales trends.

6. **Product Unit Price**
   - Compares product pricing across the product portfolio.

---

## Interactive Filters

The dashboard includes dynamic filters that update all visuals:

| Filter | Description |
|------|-------------|
| Product | Analyze specific products |
| Category | Filter by product category |
| Year | Select data from 2021–2023 |
| Quarter | Analyze quarterly sales |

## Business Insights

This dashboard helps answer questions such as:

- Which **product categories generate the most revenue?**
- What is the **sales distribution between B2B and B2C customers?**
- Which **products perform best in sales?**
- How are **discounts distributed across categories?**
- What are the **quarterly sales trends?**
